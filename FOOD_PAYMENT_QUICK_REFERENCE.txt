# 🍔 Food Payment System - Quick Reference

## ✅ Payment System Now Fully Working!

### What Gets Validated:

**✓ UPI Payment Reference:**
```
Valid:         Invalid:
✅ UPI123456   ❌ UPI12 (too short)
✅ TXN-12345   ❌ UPI_123 (underscore)
✅ PAY-APR     ❌ UPI@123 (@ symbol)
✅ foodpay-1   ❌ UPI 123 (space)
```

**✓ QR Code Reference:**
```
Valid:           Invalid:
✅ QR-PAYMENT    ❌ QR (too short)
✅ QR123456789   ❌ QR#PAYMENT (# symbol)
✅ pay-qr-001    ❌ PAY_QR (underscore)
```

**✓ Debit Card Details:**
```
Card Number:    ✅ 1234567890123456 (exactly 16)
                ❌ 123456789 (too short)

Expiry:         ✅ 03/26 (MM/YY format)
                ❌ 2603 (wrong format)

CVV:            ✅ 123 or 1234 (3-4 digits)
                ❌ 12 (too short)

Card Ref:       ✅ TXN123456 (6+ chars)
                ❌ TXN12 (too short)
```

```

✓ CAN USE: Numbers (0-9)
✓ CAN USE: Hyphens (-)
✗ CANNOT USE: Underscores, spaces, symbols, special chars
```

## 📱 For Each Payment Method:

### UPI:
- Enter 6+ character reference
- Example: `UPI123456` or `TXN-PAYMENT`
- No special characters

### QR Code:
- Shows auto-generated QR code
- Enter 6+ character reference
- Example: `QR-PAYMENT` or `QRCODE123`
- No special characters

### Debit Card:
- All 5 fields must be filled
- Card number: exactly 16 digits
- Expiry: MM/YY format
- CVV: 3-4 digits
- Reference: 6+ characters, no special chars
